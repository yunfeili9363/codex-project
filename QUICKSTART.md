# 快速开始

## 1. 启动

```bash
cd "/Users/a1-6/Documents/codex project"
npm run dev
```

## 2. Telegram 里直接发

```text
帮我看看当前项目结构
```

或者：

```text
/run 帮我检查当前仓库里最重要的入口文件
```

## 3. 常用命令

```text
/status
/abort
/history
/help
```

## 4. 语音

直接发语音。  
机器人会先转写，再继续执行。

## 5. 当前版本说明

当前项目已经简化成：

- 一个 Telegram 机器人
- 一个默认工作区
- 一个通用执行入口

旧的多场景能力已经移除，不再使用：

- `daily_todo`
- `content_capture`
- `ai_news`
